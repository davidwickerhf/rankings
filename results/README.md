# Results Directory

This directory contains the active generated outputs that support the
reproducibility package.

## Active Result Roots

- `results/fixed-merged-subarticles-edges/`
  - Fixed centrality-enriched result bundles retained in-repo.
  - These include the `total_df.csv` tables consumed directly by the
    paper-facing analysis scripts under `scripts/analysis/`.
- `results/analysis/`
  - Outputs written by the active paper-facing analysis scripts.
  - This includes performer summaries, composite-threshold evaluations,
    network-type comparisons, low-relevance-priority tests, priority
    comparisons, and overlay figures.
- `results/load/`
  - Outputs written by the active load-stage scripts when those are run.

## Supporting Outputs

- `results/analysis/00_supporting_tables/network_statistics_updated.csv`
  - Generated by `scripts/generate/01_generate_network_statistics.py`.
- `results/analysis/00_supporting_figures/correlation_matrix.png`
  - Generated by `scripts/generate/03_generate_correlation_matrix.py`.

## How Results Relate To The Scripts

- `scripts/load/` writes load-stage derived artifacts to `results/load/`.
- `scripts/generate/02_build_centrality_results.py` writes per-network
  centrality bundles such as `total_df.csv`.
- `scripts/analysis/` reads the fixed centrality bundles and writes the
  manuscript-facing outputs to `results/analysis/`.

## Archived Results

Older result families that are not part of the active paper-traceable path have
been moved to `archive/results/`.
