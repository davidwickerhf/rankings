# Documentation

This directory contains the submission-facing documentation for the repository.

- `paper/paper.tex`
  - The manuscript source kept in-repo.
- `TRACEABILITY.md`
  - Maps paper figures, tables, and claims to the scripts and generated result
    files that support them.
- `REPRODUCE.md`
  - Explains how to rerun the active scripts and use the top-level `Makefile`.
- `REFLECTIONS.md`
  - Preserves project reasoning that informed the final script-first package.

Internal drafting notes, rebuttals, and repository-agent instructions have been
moved to `archive/docs/` so that the active `docs/` tree remains focused on
supplementary materials.
