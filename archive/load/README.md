# Archived Load Material

This directory contains the older load-stage scripts and diagnostics that were
used before the active load logic was reorganized under `scripts/load/`.

## What Lives Here

- Legacy `load.py`, `process.py`, and `edges.py` variants.
- Saved descriptive outputs about unresolved references under `analysis/`.
- Historical intermediate logic that is useful for traceability but no longer
  serves as the maintained entry point.

## Current Source Of Truth

Use `scripts/load/` for active metadata processing and citation extraction.

The material here is retained to show how the project evolved, not as the
recommended way to reproduce the paper.
